## Installation

### 1. Create a new Hugo site

```bash
hugo new site mysite
cd mysite
git init
```

### 2. Add Blowfish theme as a submodule

```bash
git submodule add -b main https://github.com/nunocoracao/blowfish.git themes/blowfish
```

### 3. Copy theme configuration files

```bash
cp themes/blowfish/config/_default/* config/_default/
```

### 4. Install Emacs packages

Using `straight.el` (configured in `orgx/orgx-export.el`):

```
M-x straight-pull-all
```

Verify ox-hugo is loaded:

```
M-: (featurep 'ox-hugo)  ; => t
```

## Writing Workflow

All posts are written in **Org-mode** and exported to Markdown via ox-hugo.

### File structure

```
content-org/
├── about.org
└── posts/
    └── my-post.org       ← write here

content/                  ← generated by ox-hugo (do not edit manually)
├── about/
│   └── index.md
└── post/
    └── my-post/
        └── index.md
```

### Org file front-matter example

```org
#+TITLE: My Post Title
#+DATE: <2026-03-15>
#+HUGO_BASE_DIR: ~/Documents/devel/repos/mysite
#+HUGO_SECTION: post
#+HUGO_TAGS: hugo emacs
#+HUGO_CATEGORIES: tech
```

### Export commands

| Action | Keybinding | Description |
|--------|-----------|-------------|
| Capture new post | `C-c c b` | Create post via org-capture |
| Export current subtree | `C-c C-e H H` | Export subtree to Markdown |
| Export all subtrees | `C-c C-e H A` | Export entire Org file |

## Local Development

### Start dev server (with drafts)

```bash
hugo server --buildDrafts --disableFastRender
```

Open: [http://localhost:1313](http://localhost:1313)

### Update Blowfish theme

```bash
git submodule update --remote --merge themes/blowfish
git add themes/blowfish
git commit -m "chore: update blowfish theme"
```

### Useful Hugo commands

```bash
# Build production site
hugo

# Check for broken links
hugo --gc --minify

# Show Hugo version
hugo version
```

## Configuration

Configuration files live in `config/_default/`:

| File | Purpose |
|------|---------|
| `hugo.toml` | Site-wide settings, taxonomies, output formats |
| `params.toml` | Blowfish theme parameters |
| `languages.en.toml` | Language, author, and social links |
| `menus.en.toml` | Navigation menus |
| `markup.toml` | Goldmark renderer and syntax highlight |
| `module.toml` | Hugo module / version constraints |

## Deploy

### GitHub Actions (recommended)

```yaml
# .github/workflows/deploy.yml
- uses: actions/checkout@v4
  with:
    submodules: true
    fetch-depth: 0

- name: Setup Hugo
  uses: peaceiris/actions-hugo@v3
  with:
    hugo-version: '0.157.0'
    extended: true

- name: Build
  run: hugo --minify
```

### Manual build

```bash
hugo --minify
# Output is in ./public/
```
