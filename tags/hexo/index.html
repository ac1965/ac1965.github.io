Hexo でハロー

28 Apr 2015, 08:49
hexo

Welcome to Hexo! 環境 OS X Yosemite 10.10.3 Node.js $ git clone git://github.com/creationix/nvm.git ~/.nvm $ test -f ~/.nvm/nvm.sh && source ~/.nvm/nvm.sh $ nvm ls-remote $ nvm install 0.12.2 $ node -v v0.12.2 前準備 github でリポジトリほっておく。これは空っぽでよい。 例： Macのbrewでhubをインスコしてた場合 cd ~/Documents/github mkdir ac1965.github.io cd ac1965.github.io hub init hub create ac1965/ac1965.github.io Hexo インストール $ npm install -g hexo 準備（ブログ用のディレクトリを作る） hexo init フォルダ フォルダ指定がないと、カレントに作るのでフォルダ指定しておこう。 cd ~/Documents $ hexo init site $ cd site $ npm install 動作確認 $ hexo server この状態でウェブ・ブラウザからhttp://localhost:4000で確認してみる。
